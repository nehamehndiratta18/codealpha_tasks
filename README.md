# 📌 CodeAlpha Internship – C Programming Projects

## 📖 Overview

This repository contains three C programming projects developed as part of the **CodeAlpha Internship**.

Each project demonstrates core programming concepts such as functions, arrays, structures, file handling, and modular programming using the C language.

---

## 📁 Projects Included

### 🧮 1. Calculator Program
A menu-driven calculator that performs basic arithmetic operations using functions and switch-case statements.

### 📊 2. Matrix Operations Program
A program that performs matrix addition, multiplication, and transpose using 2D arrays and functions.

### 🎓 3. Student Management System
A file-based system that manages student records using structures and file handling with full CRUD operations.
